## Usage

```
keygen [-m|-n|-c] (optional)[-l <length>] (optional)[-f <filepath.txt>] (optional)[-g <# keys to gen>] [serial]
```

### Example

```bash
keygen -c S000Y00000000 -l 10 -f ciao.txt -g 10

// output at ciao.txt

KNB3YFY77E
TDA4LN337X

```
