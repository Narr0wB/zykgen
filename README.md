## Usage

```
usage: [-m|-n|-c] (optional)[-l <length>] (optional)[-f <filepath.txt>] (optional)[-g <# keys to gen>] (optional)[-t <threads, default is max_threads>] [serial]
```

### Example

```bash
keygen -c -l 10 -f test.txt -g 10 S000Y00000000
```
